# dotfiles
Personal Dotfiles

## Step
1. Clone repo
    ```sh
    git clone git@github.com:JeHuiPark/dotfiles.git ~/.dotfiles
    ```
1. Create symlinks
    ```sh
    ln -s ~/.dotfiles/.gitconfig ~/.gitconfig
    ```
1. Install Package
    ```sh
    brew bundle --file ~/.dotfiles/Brewfile
    ```
